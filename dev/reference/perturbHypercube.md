# perturbHypercube

Perturb the points within the sampling hypercube as per Saltelli,
Tarantola & Chan, 1999.

## Usage

``` r
perturbHypercube(hyperCube)
```

## Arguments

- hyperCube:

  Hypercube of points in parameter space at which model will be
  evaluated.

## Value

A perturbation to the points of the original `hypercube`.
