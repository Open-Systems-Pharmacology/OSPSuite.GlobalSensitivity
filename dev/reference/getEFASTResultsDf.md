# getEFASTResultsDf

Format the EFAST results into a data frame.

## Usage

``` r
getEFASTResultsDf(fftEvaluationsList, outputs)
```

## Arguments

- fftEvaluationsList:

  List of evaluated first order and total effect indices.

- outputs:

  List of `SAOutput` objects.

## Value

FFT results in data frame format.
