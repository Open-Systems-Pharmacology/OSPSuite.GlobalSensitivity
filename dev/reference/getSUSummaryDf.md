# getSUSummaryDf

Function to summarize and categorize sensitivity and uncertainty
analysis results in a table.

## Usage

``` r
getSUSummaryDf(su)
```

## Arguments

- su:

  Sensitivity and uncertainty analysis results

## Value

A list of data frames, one for each output/PK parameter combination,
where each table categorizes the sensitivity and uncertainty of each
parameter as high/medium/low.
