# addLabelsSUDF

Rename headings in summary table of sensitivity and uncertainty analysis
results.

## Usage

``` r
addLabelsSUDF(df, output, pkParameter)
```

## Arguments

- df:

  Table of sensitivity and uncertainty analysis results.

- output:

  String. Output name.

- pkParameter:

  String. PK parameter name.

## Value

Formatted table of sensitivity and uncertainty analysis results.
